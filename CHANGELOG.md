# Changelog

このファイルは agentdebuglog_parsetool のバージョン履歴と変更概要を管理します。

## v2.3

- AI Log Viewer の左上タイトル表示をボタン化し、押下時にカレンダー画面（`/`）へ遷移する導線を追加
- ブラウザ更新（`Ctrl+R` 相当）を実行する「更新」ボタンを AI Log Viewer タイトルの横に追加
- 上記ボタン導線をセッション詳細・セッション一覧・カレンダーの3画面で統一

## v2.2

- ツール実行行のヘッダ表示を強化し、ツール名の右側に仕様定義済みラベルの値のみを表示する機能を追加
- 対応ツール: `grep_search`、`memory`、`list_dir`、`apply_patch`、`read_file`、`file_search`、`create_file`

## v2.1

- モデル利用統計の表示で、`Cache xx.x%` と `out xxxx` のみを対象に閾値ベースの段階色付けを追加（確認・注意・警告）

## v2.0

- トップURL `/` を日付選択ラッパーページ化し、直近30日から `/?date=YYYY-MM-DD` へ遷移できる導線を追加
- 日付選択ラッパーページに、日付ごとのセッション件数表示（N件）を追加
- 日付選択ページを月間カレンダー表示に変更し、`/?month=YYYY-MM` で月切替できるように改善
- `0件` の日付をグレーアウトしてクリック不可に改善
- 月間カレンダーの日セルに、日次 AI credit 合計（例: `12.4 AIU`）表示を追加
- 月間カレンダーに月累積 AI credit 合計（例: `XXX.X AIU / 月`）表示を追加
- セッションの日付帰属を `session_title.datetime` から `main.jsonl` の mtime に変更し、datetime 未記録セッションの credit 脱落を解消
- セッション一覧ヘッダに「日付選択へ」リンクを追加
- index起動時に、index日付以外への遷移は live 収集へフォールバックする挙動を追加

## v1.1

- agent_response の response が途中で切れた JSON 文字列でも、末尾の文字列終端と括弧を補完して再パースできるように改善
- 上記の補完ロジックを title 抽出時にも適用し、不完全な title ログでも text part を拾いやすく改善
- アプリのバージョン管理をルートの VERSION ファイルへ集約
- セッション一覧画面とセッション詳細画面のヘッダにアプリ version を表示
- session の filtered 判定について、パスと理由を index 生成ログに出力し、後から確認しやすく改善

## v1.0

- 初期版
- JSONL の主要イベント抽出、セッション一覧表示、詳細表示、ZIP ダウンロードを提供
